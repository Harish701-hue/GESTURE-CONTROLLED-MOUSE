This project uses hand gestures to control the mouse cursor and system volume using a webcam and MediaPipe.

## Features
- Control mouse pointer with finger gestures
- Increase/decrease volume using thumb + index gestures
- Works with OpenCV and MediaPipe

## Requirements
- Python 3.x
- OpenCV
- MediaPipe
- PyAutoGUI

## How to Run
```bash
python gesture_mouse.py
